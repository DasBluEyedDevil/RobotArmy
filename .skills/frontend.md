# Frontend Skills (The Visual Hands)
**Agent**: Codex (or Cursor in visual mode)
**Specialty**: React, Vue, CSS, Tailwind, UI Components

## Skill: `implement_ui`
**Description**: Create or modify visual components.
**Usage**:
```bash
python bridge.py codex code --task "{ui_task_description}" --files "{target_file}" --reqs "{design_requirements}"
```

## Skill: `fix_visual_bug`
**Description**: Fix layout, CSS, or rendering issues.
**Usage**:
```bash
python bridge.py codex code --task "Fix visual bug: {bug_description}" --files "{file}" --context "{screenshot_description_or_error}"
```

## Skill: `refactor_component`
**Description**: Modernize or clean up a UI component.
**Usage**:
```bash
python bridge.py codex code --task "Refactor to use Hooks/Functional Components" --files "{file}"
```
